pkcs8
===
pkcs8 implements functions to process private keys in PKCS#8 format, as defined in [RFC5208](https://tools.ietf.org/html/rfc5208) and [RFC5958](https://tools.ietf.org/html/rfc5958). It can handle both unencrypted PKCS#8 PrivateKeyInfo format and EncryptedPrivateKeyInfo format with PKCS#5 (v2.0) algorithms.


[**Godoc**](http://godoc.org/github.com/youmark/pkcs8)

## Installation

```text
go get github.com/youmark/pkcs8
```
